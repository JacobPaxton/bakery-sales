# Overview
This project analyzes a Korean bakery's sales data, runs statistical tests for locality and product differences, and performs time-series analysis and projection for total sales.

# Link to data
https://www.kaggle.com/hosubjeong/bakery-sales

# Plan
1. Acquire data
2. Plot distributions of total sale amounts, products, localities
3. Split data for time-series modeling
4. Explore bivariates
5. Perform statistical testing
6. Build models for in-sample time series data
7. Compare model prediction performance for out-of-sample (future) data
8. Select best model
9. Test model performance against final out-of-sample data
10. Report findings
